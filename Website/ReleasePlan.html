ACME Release Plan

(Updated Nov 24, 2014 while Iteration 5)
           On Saturday (20th September), we met as a group to discuss and plan our project. We started by going over both librarian and researcher user stories. Together we came up with 22 user stories combined. We then took time to group user stories into features that could be developed over the same time period. We used HIGH/MEDIUM/LOW method to prioritize features described in the 22 user stories. In the Commitment Phase we have come up with the number of days each user story will take to function which totals to 50 days for entire project. However, we found that amount of hours is more accurate in planning than days. Therefore starting from Iteration 3, we are going to estimate in hours. Every Phase we will reiterate by improving our application and adding new features using the agile method.
