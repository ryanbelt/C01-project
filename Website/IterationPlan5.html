Iteration Plan 5

Having completed Iteration 4, we moved on to the remainder of our user stories. These user stories were features which were not necessary, but would be nice to have. This involves access to manually modify minor settings, delete data through terminal and a help button. These features were from user stories 6, 10 and 20 respectively.

As a group we decided that these user stories although not necessary, would be nice features to add which could be beneficial to our clients. However, given their low priority there would be little disappointment if not all the user stories were completed. We have sorted the user stories by Priority, then by Cost.

Total User Stories: 3
Total Cost: 12 Hours
Start: November 19th 2014
End: November 26th, 2014

Project Velocity


The project velocity for this iteration was 12 hours over the iteration of 7 days. This is lower as this is our final iteration, and we have very little user stories left. In the end we only ended up finishing user story 10 and 6.
